# AXIOMTEK 報價單產生器

艾訊 (AXIOMTEK) 業務用的網頁版報價單產生器。在瀏覽器中直接填寫、計算並列印 / 另存 PDF 的報價單，同時維護一份可跨裝置同步的「常用料號清單」。

目前為單一 HTML 檔的雛形版本，所有 HTML / CSS / JavaScript 皆內嵌於同一個檔案中。

## 功能

- **報價單編輯**：品牌表頭、客戶 / 業務資料、注意事項、幣別、備註、簽署欄，多數欄位可直接於頁面上編輯。
- **品項表格**：動態增減品項列，自動計算單項總價、合計、稅額 (5%) 與總計。
- **成本 / 毛利分析**：每列顯示成本與毛利 (GP%、每件毛利金額，負毛利以紅色標示)，底部彙總成本合計、毛利與毛利率。
  - 成本 / 毛利欄位**僅在螢幕上顯示，列印 / 另存 PDF 時會自動隱藏**，避免內部成本外流。
- **常用料號清單**：報價時輸入完整料號 / 品名 / 規格會自動記錄；再次輸入同一料號可自動帶出品名、規格、單價與成本。
- **匯入 / 匯出**：
  - 匯入既有料號資料 (Excel `.xlsx` / `.xls` / CSV)，含欄位自動對應與確認介面。
  - 常用料號清單可匯出 / 匯入 JSON 備份。
- **雲端同步**：透過 Firebase 帳號登入後，常用料號清單即時同步至 Firestore，跨裝置共用；未登入時退回瀏覽器本機儲存 (localStorage)。
- **列印 / PDF**：透過瀏覽器列印功能輸出，並針對列印調整版面。
- **響應式版面**：螢幕寬度 760px 以下自動切換為單欄排版。

## 使用方式

線上版本：<https://axiomtektw.github.io/quote/>

亦可直接以瀏覽器開啟 `index.html` 於本機使用。

雲端同步與 Excel 匯入功能需要網路連線 (相依套件透過 CDN 載入)。

## 技術

- 單一 HTML 檔，無需建置流程。
- [SheetJS (xlsx)](https://sheetjs.com/) — Excel / CSV 讀取。
- [Firebase](https://firebase.google.com/) (Auth + Firestore) — 帳號登入與雲端同步。
- 資料儲存：未登入使用 `localStorage`；登入後使用 Firestore。

## 待辦 / 已知事項

- 報價單本體內容 (客戶、品項) 目前尚未提供存檔 / 歷史紀錄功能，重新整理頁面後即清空。
- Firestore Security Rules 需確認已正確設定，以保護料號成本等內部資料。
- 稅率目前固定為 5%。
